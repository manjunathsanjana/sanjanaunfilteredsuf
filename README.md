# Sanjana Unfiltered

A content-driven lifestyle blog exploring food, travel, and personal stories—now powered by **Astro v6**.

## Features

- **Astro v6 Architecture:** High-performance, components-based structure with Zero-JS by default.
- **Content Layer API:** Managed via Markdown files in `src/content/blog/` for easy updates.
- **View Transitions:** Smooth, app-like page transitions without hard refreshes.
- **Modern Responsive UI:** Premium desktop experience with optimized mobile interactions.
- **Clean Social Integration:** Featuring animated X (Twitter) profile links.

## Tech Stack

- **Framework:** [Astro 6.1.3](https://astro.build)
- **Styling:** Vanilla CSS (Modern CSS variables and flex/grid layouts)
- **Content:** Markdown (Markdown + YAML Frontmatter)
- **Deployment:** Static Site Generation (SSG)

## Local Development

To run the blog locally, ensure you have [Node.js](https://nodejs.org/) installed, then:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

Once running, navigate to `http://localhost:4321` in your web browser.
